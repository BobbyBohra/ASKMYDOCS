# 🤖 AskMyDocs — Chat with your PDF!

A RAG-based intelligent PDF chatbot built with Groq LLM and ChromaDB.
Upload any PDF and ask questions in natural language!

## 🚀 Features
- Upload any PDF document
- Ask questions in natural language
- Get instant AI-powered answers
- View source references
- 100% Free (Groq API)

## 🛠️ Tech Stack
- **LLM:** Groq (LLaMA 3.3)
- **Vector DB:** ChromaDB
- **Frontend:** Streamlit
- **PDF Processing:** PyPDF
- **Language:** Python

## ⚙️ Installation

1. Clone the repo
   git clone https://github.com/BobbyBohra/AskMyDocs-RAG-Chatbot.git

2. Install dependencies
   pip install -r requirements.txt

3. Create .env file
   GROQ_API_KEY=your_api_key_here

4. Run the app
   streamlit run app.py
